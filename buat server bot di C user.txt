Langkah 1: Buka Terminal / CMD

Buka aplikasi Terminal (Mac/Linux) atau Command Prompt/PowerShell (Windows), lalu buat folder baru dan masuk ke dalamnya:
Bash

mkdir wa-notifier-bot
cd wa-notifier-bot

Langkah 2: Inisialisasi Project Node.js

Jalankan perintah ini untuk membuat file konfigurasi package.json otomatis:
Bash

npm init -y

Langkah 3: Install Library yang Dibutuhkan

Anda wajib menginstall 3 library utama yang tertulis di kode tersebut (whatsapp-web.js, qrcode-terminal, dan express). Jalankan perintah ini di terminal:
Bash

npm install whatsapp-web.js qrcode-terminal express

Langkah 4: Buat File server.js

    Buat file baru bernama server.js di dalam folder tersebut.

    Buka file itu menggunakan text editor (seperti VS Code atau Notepad++).

    Copy-paste seluruh kode yang Anda berikan di atas ke dalam file server.js tersebut, lalu simpan (save).

Langkah 5: Jalankan Bot WhatsApp

Di terminal Anda, ketik perintah berikut untuk menyalakan servernya:
Bash

node server.js
