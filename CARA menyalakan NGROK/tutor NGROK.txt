Cara Membuka Kembali Ngrok
Buka kembali CMD (Command Prompt) di laptopmu.
Masuk ke folder tempat file ngrok.exe kamu berada (atau jika sudah masuk Environment Path, kamu bisa langsung ketik perintahnya).
Jalankan perintah untuk mengarahkan ke port server.js (port 3000):

Bash

ngrok http 3000

Ngrok akan terbuka dan memberikan alamat URL baru yang acak di baris Forwarding, contohnya seperti:https://a1b2-34-56-78.ngrok-free.dev

Langkah yang Wajib Dilakukan Setelah Mendapat URL BaruKarena alamatnya berubah, kamu harus memperbarui "alamat tujuan" di file PHP hosting kamu agar tembakan datanya tidak nyasar ke alamat yang lama.Copy alamat https://xxxx-xx-xx...ngrok-free.dev yang baru dari CMD Ngrok.

Buka file take_order.php di hosting InfinityFree.Scroll ke paling bawah, 
cari fungsi kirimKeBotLokal.Ubah baris CURLOPT_URL dengan alamat yang baru tadi:PHP// Ganti dengan URL Ngrok baru yang sedang aktif saat ini
CURLOPT_URL => 'https://alamat-ngrok-baru-kamu.ngrok-free.dev/api/send-notification',

Save file take_order.php.

💡 Tips Tambahan: Cara Mengunci Alap-Alap Ngrok agar Tidak Berubah (Tetap Gratis!)

Ngrok sebenarnya mengizinkan pengguna akun Free untuk memiliki 1 buah Static Domain (Domain Tetap) secara gratis. Dengan cara ini, domain Ngrok kamu tidak akan pernah berubah lagi meskipun CMD ditutup berkali-kali!Cara klaim domain tetap gratis dari Ngrok:

Buka browser dan login ke dashboard kamu di dashboard.ngrok.com.
Di menu sebelah kiri, cari menu Cloud Edge $\rightarrow$ Domains.
Di sana akan ada opsi untuk membuat 1 domain gratis (biasanya berformat nama-acak.ngrok-free.app atau sejenisnya). Klik Create Domain.Setelah dapat domain tetap itu, cara menyalakan di CMD laptop kamu berubah menjadi:Bash

ngrok http --url=schedule-oblivion-dastardly.ngrok-free.dev 3000
